快捷方式

1. Tab：补全（命令、文件名、路径）
    敲一下补全，敲两下显示更多
2. 上下键：向上/向下翻找历史命令
3. clear / ctrl + L：清屏
4. ctrl + shift + n / t : 新建终端 / 新建标签
5. alt + Tab ：切换
