最近正在做一个机械臂的项目，要用到Linux，OpenCV等多项技术，且开发学习过程中代码也比较多，便想尝试将这个项目的代码上传到Github上。一方面方便自己学习、整理项目中用到的代码；另一方面日后从事相关专业Github是逃不开的。

写这篇文章主要目的是为了记录下把项目代码上传至github的过程，以免以后忘记了又要去找。如果你恰巧需要且翻到这篇博客，大可看一看，整个过程仅需几条指令，并不繁琐。
