s:套接字（socket）

快捷方式

1. Tab：补全（命令、文件名、路径）
    敲一下补全，敲两下显示更多
2. 上下键：向上/向下翻找历史命令
3. clear / ctrl + L：清屏
4. ctrl + shift + n / t : 新建终端 / 新建标签
5. alt + Tab ：切换

编辑器

1. gedit：

类似Windows下的记事本，图形界面
进入编辑模式

在终端下不退出gedit模式时无法在同一个终端下进行其他操作

2. vi/vim：

非图形界面，靠命令操作
进入编辑模式：

三个模式

  • 命令行模式
    用户打开vi进入的模式，可对文件进行批量处理
    复制 删除 粘贴 剪贴 撤销 反向撤销
    ctrl + r
  • 插入模式
    文本录入
    进入插入模式：
    a：从光标所在后一个位置进行录入
    i：从光标所在的位置进行录入
    o：从光标下一行进行录入
esc退出插入模式
  • 底行模式
    注意：底行模式下所有命令以(:)开头
    保存 退出 强制退出 保存并退出 复制 删除

物联网

项目简介

一种帮助完成游泳训练计划的游泳智能训练系统，其特征在于：

  1. 使运动员自行或由教练员在移动设备上添加或分享计划项目及各项目包干时间，并下载于终端设备上。
